⚙️ One-time setup — connect your Google Sheet

  1. Open your AcadoreSkills Jobs Google Sheet
  2. Click File → Share → Publish to web
  3. Choose Sheet: Jobs → Format: CSV → Click Publish
  4. Copy the URL it gives you
  5. Open this HTML file in a code editor
  6. Find SHEET_CSV_URL below and paste your URL there
  7. Save and reload — live jobs will appear instantly!
